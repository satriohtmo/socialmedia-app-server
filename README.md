Captiverse Backend

Tech Stack :

- Nodejs
- Sequelize (ORM)
- Postgres (DataBase)
- Multer (Photo Upload)
- Bcrypt (Hashing Password)
- JsonWebToken (Token)

To run this server :

- create file .env
- set config reference at .env.example
- npx sequelize db:create
- npx sequelize db:migrate
- npm run start
